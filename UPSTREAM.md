# Upstream Origin

DeskPulse started as a fork of:

https://github.com/HermannBjorgvin/Clawdmeter

The original project focuses on a simple Claude Code usage desk toy. This fork
is maintained separately for multi-provider usage monitoring, currently Claude
Code and Codex.

## Fork Status

This repository intentionally remains in GitHub's fork network so that the
upstream origin stays visible in the GitHub UI. If the repository is detached in
the future, keep this file and the README attribution so the origin remains
clear.

## Direction

The upstream maintainer clarified that the original project is intended to stay
focused on Claude usage as a simple desk toy. DeskPulse is a separate direction
for a desk-side AI usage monitor that can show multiple providers.
